Frequently Asked Questions:


1. Do I need to register to use the contents of the website?

For now yes. We are hoping to make all the content available to everyone without a register option.

2. Can I change my personal information?

Yes, you need to go to settings page for that.

3. Where do I find the lessons?

You need to select a topic to find lessons in it.

4. Why can't I sign in?

You need to sign up before signing in or need to provide correct username and password.

5. Does it accept UTSA id for sign in?

By default no, you need to signup before doing that

6. Is username is email id?

No, username and email id are different, to sign in username must be provided.

7. How does search feature work?

If you are in topics page it will result in topics otherwise it will show the lessons available.

8. What is bookmark button?

If you bookmark any topic or lesson it will be available under bookmarks page where you can quickly access the information.
